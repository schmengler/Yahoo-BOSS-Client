		garbageCollector (line 99)
	
 
	
	

Deletes expired cache files. It can check all cache files which may be resource expensive or just look for old files and stop after a specific number of files are deleted.

	
    
				
  • access: public
    • 
			

	
	

		void
		
			garbageCollector
		
					([ $olderThan = null], [int $limit = null])
			

	
			
    
					
  • 
				\DateInterval
				$olderThan: if set, only cache files older than this interval will be checked to save cpu and filesystem usage			
    • 
					
  • 
				int
				$limit: if set, the garbage collector stops after deleting $limit files. This is useful if you run it within every nth user request.
